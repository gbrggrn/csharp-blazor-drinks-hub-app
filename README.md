\# DrinksHubApp



\## Context

\* \*\*Origin\*\* Developed as part of the third C# course at Malmö University.

\* \*\*Objective\*\* This web application acts as a presentation/review site for drinks.

\* \*\*Status\*\* 🟢 Complete/Functional



---



\## Systems Architechture

\* \*\*Logic\*\* Blazor Server-side Rendering + Web API

\* \*\*Tech Stack\*\* C# + .NET + Blazor + Web API + EF Core + SQLite

\* \*\*Testing\*\* Swagger for API endpoint testing



---



\## Functionality

\* Role based access

\* Users not logged in can:

&nbsp; - View drinks info and reviews

\* Users logged in can:
  - Review drinks and add favorites
\* Admin users can:
  - Add, remove and edit drinks

---



\## Setup \& Usage

1\. Clone the repository

2\. Open in IDE of your choice

3\. Create your own universe of drinks.



---



\## Learning Outcomes

\* Async programming

\* Basic HTML with bootstrap shorthand

\* Blazor setup and basics
\* Basic role based access
\* Web API setup
\* Testing API endpoints with swagger
